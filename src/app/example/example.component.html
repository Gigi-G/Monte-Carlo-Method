Now, we want to calculate the area of ​​the Lake Garda using the Monte Carlo method, so we need to generate X random dots within the image. The results of the areas will be displayed in a chart. We, also, use two colors to distinguish the dots:

  • red for earth dots;
  • black for lake dots.

First of all, we try to calculate the lake area using only 100 dots.









We can find on Internet that the area of the Lake Garda is 370 km2.



Conclusion:

From the chart we can see that when the number of dots increase the estimated result is more similar to the real one.

I also remind you that the image is taken from Google Maps, so it's not perfect!
